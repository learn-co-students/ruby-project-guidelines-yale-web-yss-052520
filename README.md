# Reading List

Reading List is a Ruby CLI app for organizing reading lists.

## Installation

Run the bin/run.rb file to install.

```bash
ruby bin/run.rb
```

## Usage

```ruby
Hi! Welcome to your reading lists!
----------------------------------------------------------------------------------------------------
Main Menu (Use ↑/↓ arrow keys, press Enter to select)
‣ View Lists
  Create New List
  Exit Program
```
